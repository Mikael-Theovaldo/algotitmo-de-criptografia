# 🔐 Sistema de Criptografia Simples em Java

Este projeto foi desenvolvido com o objetivo de praticar e consolidar conceitos fundamentais de programação em Java, aplicando técnicas de **criptografia simples** e manipulação de strings.  

## 🚀 Funcionalidades

O sistema oferece duas formas de manipulação de mensagens:

1. **Criptografia por Substituição Simples**  
   - Substitui vogais por símbolos específicos:  
     - `a → %`  
     - `e → 8`  
     - `i → !`  
     - `o → §`  
     - `u → #`  

   Também é possível **reverter** essa criptografia, retornando a mensagem original.

2. **Inversão de Palavras**  
   - Inverte a ordem dos caracteres da mensagem.  
   - Possui opção para **reverter** e restaurar o texto original.
